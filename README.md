# Grind — Coffee for Students & Hustlers

A landing page for Grind, a coffee shop built for late nights, deep focus, and the kind of quiet that gets things done.

## Tech Stack

- Laravel 13
- Livewire v4
- Tailwind CSS v4
- Vite

## Features

- Responsive landing page
- Navbar with scroll behavior
- Hero section with animated SVG illustration
- Features section
- Menu section
- CTA section
- Contact form with Livewire validation
- Google Maps embed
- Footer

## Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/lianxxxx/offline-practice.git
cd offline-practice
composer install
npm install
cp .env.example .env
php artisan key:generate
npm run dev
php artisan serve
```
